# Project Roadmap

## Phase 1 — MVP (Current)
- Student CRUD operations
- Attendance marking with Present/Absent/Late
- Role-based authentication (Super Admin, HOD, Teacher, Student)
- Dashboard with basic statistics
- Dark mode support
- Responsive design

## Phase 2 — Intelligence & Cloud
- Google Gemini AI integration for attendance analysis
- Google Drive backup and restore
- AI chat assistant for insights
- Analytics with charts and heatmaps
- PDF export for reports

## Phase 3 — Biometrics & Scale
- Face recognition for automated attendance
- NFC/RFID card support
- Push notifications
- Parent/guardian portal
- Real-time sync across devices

## Phase 4 — Enterprise
- Persistent database (PostgreSQL)
- Multi-institution support
- LDAP/SSO integration
- Advanced reporting engine
- Mobile native apps (React Native)
- API rate limiting and caching
